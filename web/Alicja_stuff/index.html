how to do it
.01
Load the log file

W panelu administracyjnym otwórz dowolny blok statyczny (CMS -> Bloki statyczne). W edytorze naciśnij przycisk “Insert image”

W następnym okienku stwórz katalog o nazwie Logo (Create folder), wybierz plik z logiem (Browse files) i dodaj (Upload files) go do utworzonego katalogu.

.02
Configuration

Aby wskazać nową ścieżkę do logo, przejdź do zakładki
Wygląd (System -> Konfiguracja -> Ogólne -> Wygląd).
W zakładce Header (Nagłówek) znajdziesz pola do podania ścieżki logo. Twoja nowa ścieżka będzie wyglądać tak:
../../../../media/wysiwyg/logo/twojenowelogo.png
